A Unified hook for sending emails, bring your own provider and use this package to send emails. Works on Node.js, Bun, Deno and Cloudflare Workers. Typesafe and supports modern email providers.

Supported Providers

1. Resend
2. Sendgrid
3. Postmark
4. Plunk
5. Mailgun
